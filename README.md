# carte
精美小巧的移动Apps模板。

## [DEMO](https://icyse.github.io/carte/)请务必使用手机浏览，或者在PC上使用Chrome的手机浏览器模拟功能。

## 特点

* 页面跳转过场动画;
* 图片轮播组件;
* 评分组件;
* 弹出框、浮动框;
* 列表;
* 下拉刷新;
* 幻灯片组件;
* comment、feed组件;
